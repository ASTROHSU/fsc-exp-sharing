# 虛擬資產風險管理經驗分享

> 2025 年學費最貴的三堂課：Bybit 遭駭、幣安 USDe 脫鉤與 DeFi 海嘯

這是一個互動式簡報網頁，分享虛擬資產領域中三個重要的風險管理案例與實務觀察。

## 📋 專案簡介

本專案以網頁簡報形式呈現，探討虛擬資產保管商、借貸商及 DeFi 業務的風險管理注意事項。透過三個真實案例，深入分析：

1. **保管風險**：Bybit 15 億美元遭駭事件
2. **產品設計風險**：幣安 USDe 脫鉤事件
3. **DeFi 投資風險**：可組合性與流動性風險

## ✨ 功能特色

- 🎨 **現代化設計**：採用深色主題，符合 Blocktrend 品牌色彩

- 🎯 **互動式簡報**：支援鍵盤與按鈕導航
- 🔄 **流暢動畫**：頁面切換過渡效果
- 📊 **視覺化圖表**：流程圖與圖解說明

## 🚀 使用方法

### 本地運行

1. 克隆或下載此專案
```bash
git clone https://github.com/ASTROHSU/fsc-exp-sharing.git
cd fsc-exp-sharing
```

2. 直接在瀏覽器開啟 `index.html` 檔案
   - 或使用本地伺服器（推薦）：
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需安裝 http-server)
npx http-server

# 使用 PHP
php -S localhost:8000
```

3. 在瀏覽器中訪問 `http://localhost:8000`

### 操作說明

- **下一頁**：點擊右下角右箭頭按鈕，或按鍵盤 `→` 鍵、`Space` 鍵
- **上一頁**：點擊右下角左箭頭按鈕，或按鍵盤 `←` 鍵
- **頁碼顯示**：左下角顯示當前頁碼
